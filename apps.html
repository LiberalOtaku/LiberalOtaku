Recommended Apps

If you want some personal recommendations regarding which software or services to use for various things, then this page is for you! These are apps that I either use on a daily basis or chose over similar apps for various reasons, and I will soon list those reasons below as well. Some are paid apps, but most are free and open-source, which is what I mainly look for in an app wherever possible.

Quick disclaimer: I am not affiliated with or sponsored by any of the apps mentioned here. This is entirely my opinion regarding which apps I believe are either worth sharing or simply better than competing apps.

Software Development

  • Atom - free text editor for all of my software development
  • Git - the popular version control system
  • Notepad++ - free alternative to Windows Notepad for light text editing
  • Sourcetree - free Git desktop client for Windows
    (I use the command line interface while working in Linux)
  • Texmaker - free software for writing scientific and mathematical documents in LaTeX

Multimedia Tools

  • Audio
    • Ableton Live - DAW for music production
    • Audacity - free software for audio analysis and editing
    • Exact Audio Copy - free software for making digital backups of my music CD collection
    • foobar2000 - free Windows music player with a versatile layout and plenty of features
    • LilyPond - markup language for writing beautiful sheet music
    • Mixxx - free DJ mixing software with BPM and key analysis
  • Pictures
    • GIMP - the free alternative to PhotoShop
  • Videos
    • Blender - mainly used for 3D modeling, but I currently use it for video editing
    • OBS Studio - free software for streaming and screen capture recording
    • VLC Media Player - the free multimedia player

Communication

  • ProtonMail - secure email service
  • Slack - the popular app for workspace communications
  • Sylpheed - free and lightweight email client
  • Telegram - cross-platform instant messaging

Online Security

  • ExpressVPN - VPN provider
  • Master Password - the most secure offline password manager

Android Apps

  • Akebi - offline Japanese dictionary on-the-go
  • Bitdefender Antivirus Free - simple yet effective antivirus protection for Android devices
  • USB Audio Player PRO - Android music player compatible with external DACs
